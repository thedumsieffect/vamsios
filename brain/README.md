# Brain

Personal and professional intelligence, organized for AI-assisted work across all projects and contexts.

## How It Works

- **Source of truth** for who Vamsi is, how he works, who he works with, and what's happening across projects.
- **Daily sync** from Roam meeting transcripts at 6pm CT — extracts signal, routes to the right file, commits automatically.
- **Compounding** — files grow richer over time as more meetings are processed. Per-person files build a complete picture of each relationship.

## Structure

```
brain/
├── profile/          # Identity, work style, personal life
├── people/           # One file per key person — role, dynamics, patterns
├── projects/         # Active project context, decisions, direction
├── decisions/        # Monthly decision logs — date, who, what, why
├── competitive/      # Competitor and market intelligence
├── feedback/         # Coaching received, collaboration preferences
└── sync/             # Processing log — tracks what's been pulled from Roam
```

## Sync Rules

- **Included:** Meetings with ≤5 participants (1:1s, small working sessions)
- **Excluded:** Team demos, all-hands, large group calls
- **Deep processing:** Kyra 1:1s always get full transcript analysis
- **Others:** Summary + selective chapter transcripts for high-signal sections
